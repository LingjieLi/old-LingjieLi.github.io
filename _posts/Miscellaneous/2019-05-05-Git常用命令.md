1. 当前分之所指的对象
`git log --oneline --decorate`
2. 项目分叉历史
git log --oneline --decorate --graph --all
3. 创建新分支
`git branch iss53`
4. 检出/切换分支
`git checkout iss53`
5. 清除暂存区(当修改.gitignore文件时，需要执行这个命令)
`git rm -r --cached .`
